## Prequisites

Node v14.x

## Installation

- `git clone git@github.com:dOrgTech/starkware-demo.git`
- `cd starkware-demo`
- `yarn`
- Create a `.env` file at root level and ask one of the maintainers for the required API keys and URLs
- `yarn dev`
